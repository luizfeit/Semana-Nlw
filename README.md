- authenticated routes

Before starting üèÅ, you need to have [Git](https://git-scm.com) and [Node](https://nodejs.org/en/) installed.

## Starting

```bash
# Clone this project
$ git clone https://github.com/Willian-Rodrigues/NextLevelWeek-Heat
# Access
$ cd NextLevelWeek-Heat/backend-node
# Install dependencies
$ yarn or npm install
# Run the project
$ yarn dev or npm run dev
# The server will initialize in the <http://localhost:4000>
$ cd NextLevelWeek-Heat/web-react
# Install dependencies
$ yarn or npm install
# Run the project
$ yarn dev or npm run dev
# The server will initialize in the <http://localhost:3000>
```
